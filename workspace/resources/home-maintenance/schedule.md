# Home Maintenance Schedule

## Monthly
- inspect or replace HVAC filter if needed
- pantry and fridge sweep
- tidy one clutter hotspot
- review household supplies

## Quarterly
- donation run
- seasonal clothing check
- safety gear check
- under-sink or utility-area inspection

## Seasonal
- spring reset
- summer activity gear check
- fall school/home reset
- winter medicine + pantry audit
